# LMS_api_project
Cách chạy hệ thống: 
1. Truy cập đúng thư mục có requirement.txt
    cd backend_lms
2. Cài đặt thư viện : pip install -r requirement.txt
3. Tạo database có tên là db_education trong mysql 
4. Chạy migrate : python manage.py migrate (bắt buộc)
5. Chạy server: python manage.py runserver
6. Tải extension Go Live trên vscode
7. Chạy Go Live ở file index.html
